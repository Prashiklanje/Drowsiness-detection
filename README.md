😴 Driver Drowsiness Detection System

A deep learning-based Driver Drowsiness Detection System that detects whether a driver's eyes are open or closed in real time using a webcam and alerts when drowsiness is detected.

📌 Project Overview

Driver drowsiness is one of the major causes of road accidents. This project detects drowsiness by analyzing eye movements through a webcam using CNN and OpenCV.
🛠️ Tools & Technologies

- Python
- TensorFlow
- Keras
- OpenCV
- NumPy
- Matplotlib
- Scikit-learn
- Computer Vision
- Deep Learning
